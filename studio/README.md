# eml-annotation-studio

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Running Tests

Tests are powered by Vitest and React Testing Library.

To run the test suite once:
`npm test`

To run tests in watch mode:
`npx vitest`

## Linting

This project uses ESLint. To check for any linting errors, run:
`npm run lint`
